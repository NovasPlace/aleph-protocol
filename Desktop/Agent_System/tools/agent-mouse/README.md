# Tools: Agent Mouse

**Component Type**: Sovereign Engine Developer Tool
**Status**: CLI / Utility

## Role
main.py — Entry point for the Agent Mouse system (software-only mode).

Captures real mouse input via pynput, detects gesture patterns,
and executes mapped OS commands via xdotool. Shows a live terminal
dashboard with status and recognized gestures.
