# Organs: Cognitive Biases

**Component Type**: Sovereign Engine Cognitive Organ
**Status**: Active / Production

## Role
CognitiveBiases — Bias filters for the Sovereign Organism.

Re-scores memory recall results with cognitive biases:
recency, availability, and confirmation bias.

Usage:
    from cognitive_biases import CognitiveBiasEngine
    engine = CognitiveBiasEngine()
    biased = engine.apply_all(memories, mood="curious")
