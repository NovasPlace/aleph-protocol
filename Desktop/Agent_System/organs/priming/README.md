# Organs: Priming

**Component Type**: Sovereign Engine Cognitive Organ
**Status**: Active / Production

## Role
Priming — Spreading Activation for the Sovereign Organism.

Boosts related memories via the knowledge graph. Primed memories
surface first in recall, then decay back to baseline.

Usage:
    from priming import PrimingSystem
    ps = PrimingSystem()
    ps.prime("mem_abc123", boost=0.3, max_hops=2)
